Thẻ code khi kết hợp với HTML Entities sẽ hiển thị được mã code lên trình duyệt

- Có thể chèn code trong 1 đoạn văn hoặc chèn hẳn 1 đoạn code dài. Phục vụ cho những trang web dạy lập trình chẳng hạn.
- Tại sao cần phải dùng thẻ code? Nếu căn chỉnh ta vẫn có thể tạo được đoạn code từ các thẻ khác nhưng trình duyệt lại không đọc được đó là code, chỉ mắt thường hiểu thôi, muốn trình duyệt hiểu được đó là code thì ta cần đặt nó trong thẻ code, thêm thẻ pre (preformatted) để giữ nguyên trạng thái đoạn code 


      <head>
        <meta charset="UTF-8" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
        <title>Code tag in HTML</title>
      </head>
      <body>
