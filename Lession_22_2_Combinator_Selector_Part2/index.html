Hôm nay chúng ta học về gom nhóm các phần tử HTML để CSS khi chúng có cùng class name và tiếp đó học về cách chọn 1 phần tử cụ thể khi nó trùng tên class với 1 element khác.

Syntax:
1. selector0, .selector1, #selector2 {code here ...} (Áp dụng cho bộ chọn được liệt kê, mỗi bộ chọn cách nhau bởi dấu phẩy)
2. .selector1.selector2{ code here ...} (Cái này thì gần giống điều kiện and, tức là css cho phần tử có class là selector1 và selector2, chỉ HTML element nào có đủ 2 class thì mới bị ảnh hưởng)
