Learn About CSS Prefix

  1. CSS có rất nhiều thuộc tính nhưng và không phải trình duyệt nào cũng được hỗ trợ
  2. Khi gặp sự cố CSS không hỗ trợ tất cả trình duyệt, ta cần thêm tiền tố prefix. Google search vendor prefix css để biết được tiền tố của các browser
  3. Nếu như gặp các thuộc tính cần prefix thì ta không nên thêm thủ công từng cái 1 mà hãy cầm file css đưa lên google và search auto prefix rồi paste css vô cho web nó tự thêm prefix :V
  4. Vậy làm sao để biết 1 thuộc tính ta mới gặp lần đầu nó có cần sử dụng prefix hay không? Sử dụng trang web caniuse.com để biết nhe.
