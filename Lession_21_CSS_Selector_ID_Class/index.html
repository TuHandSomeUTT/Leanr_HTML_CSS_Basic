CSS Selector là gì?

1. HTML là ngôn ngữ đánh dấu, CSS để tạo kiểu, khi viết web lớn sẽ có hàng trăm, hàng ngàn phần tử HTML. Vậy làm thế nào để xác định chính xác phần tử nằm trên trang đó để tác động chính xác lên phần tử đó thôi? Ta sử dụng CSS Selector nhé

2. CSS Selector hay còn gọi là bộ chọn, chỉ đơn giản là nó giúp ta chọn đúng phần tử nằm trên trang HTML

3. id selector: Dùng khi chỉ muốn áp dụng CSS cho 1 phần tử chỉ định nào đó của HTML thì khi đó ta dùng id. Tình huống sử dụng là khi ta có nhiều phần tử HTML giống nhau nhưng ta chỉ muốn tác động vào 1 phần tử cụ thể nào đó thì lúc đó ta dùng id. Không nên đặt id trùng tên nhau để tránh bị rối.

4. Class selector: Do id được sử dụng lại tên id. Trong trường hợp muốn khai báo lại thì ta dùng class selector

Khi sử dụng id selector thì ở trong phần tử html ta gõ id và đặt tên id trong cặp dấu "" sang bên file css với cú pháp bắt đầu bằng dấu #tenid

class selector thì quy tắc đặt tên cũng như id selector, chỉ khác bên css bắt đầu bằng dấu . thay vì dấu #. Example: .tenClass
1 phần tử HTML có thể nhận nhiều class css, các class các nhau bằng dấu space.
