What is Combinator Selectors ?

Trên web pages có nhiều phần tử HTML, nếu ta dùng id và class đơn thuần sẽ làm code trở nên nặng nề hơn do có nhiều phần tử có chung thuộc tính css. Do vậy nên có bộ chọn kết hợp để gom nhóm lại giúp code ngắn hơn.

Hôm nay ta sẽ học
  • * selector
  • element .class

* selector để tác động lên tất cả các thẻ, phần tử HTML trên web pages.
CSS syntax: * { code here}

Chẳng hạn ta muốn chọn tất cả phần tử h3 với class là intro thì syntax như sau: (Đây là trường hợp các tag dùng chung class, và các tag khác nhau)
- Bên file HTML phải đặt tên class trước đã :V
- Bên file CSS: h3.intro {Code here ...}

Túm lại bài này gom các element có cùng kiểu class với nhau để dễ dàng CSS. Nếu gặp cái nào cần CSS riêng thì dùng element.class
