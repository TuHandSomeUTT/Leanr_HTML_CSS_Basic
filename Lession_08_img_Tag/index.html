img là thẻ tự đóng, dòng để hiển thị hình ảnh trên web pages

img syntax:
- Thuộc tính src(source): Dùng để chỉ định đường dẫn tới tệp tin hình ảnh.
- Thuộc tính alt(alternative - Thay thế): Dùng để hiển thị text khi link ảnh bị lỗi, hỗ trợ screen reader, SEO
- Thuộc tính width: Dùng để chỉnh chiều rộng của ảnh
- Thuộc tính height: Dùng để chỉnh chiều cao của ảnh
- Nếu không tùy chỉnh rộng, cao thì hình ảnh sẽ được hiển thị kích thước mặc định
- Nếu chỉ set 1 trường là width hoặc height thì chiều còn lại sẽ tự động căn sao cho vừa với trang web
